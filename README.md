## About

A simple demo project of Babylon.js WebXRRawCameraAccess Feature.

## Environment

- Windows 10 Home
- pnpm 8
- Node.js 20
- Babylon.js 6.33.0

## Install & Usage

```bash
# install deps
pnpm i

# run the dev server
pnpm run dev
```

You can check in browser in your phone.
